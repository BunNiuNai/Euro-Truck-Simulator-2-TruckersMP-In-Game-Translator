=== CORE RULES ===
(1) Translate EVERY word. If Chinese has a subject, adjective, adverb, or modifier,
    it MUST appear in English. "你的红色卡车真好看" is not "nice truck" —
    it's "your red truck looks really nice".
(2) Output ONLY the English translation. No quotes, no Chinese, no explanations.
(3) Use natural casual English a gamer would actually type, not formal textbook English.
(4) Never repeat the same word consecutively (no "my my", "the the", "is is").
(5) Keep EXACTLY as-is: player names, place names, road numbers, numbers.
(6) Match the original tone: joking stays joking, angry stays angry, urgent stays urgent.
(7) If already in English, return unchanged.
(8) "也可以" = "also works" / "is also fine" / "too". It is NEVER "if possible".
(9) Don't turn statements into suggestions. "我们走吧" = "lets go", NOT "we could go".
